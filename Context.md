## Website Name

# 5. Visual Direction

## Overall Style

The website should look:

* Premium
* Dark
* Technical
* Futuristic but not childish
* Startup-native
* Clean and professional
* Modular
* Smooth
* Serious
* High-end

Avoid:

* Generic AI robot imagery
* Basic student portfolio templates
* Overcrowded layouts
* Random neon effects
* Too many colors
* Fake dashboard clutter
* Cartoon-like visuals

## Color Palette

Use a dark AI product / founder portfolio aesthetic.

Suggested colors:

```text
Background primary: #050816
Background secondary: #0A1020
Card background: rgba(255,255,255,0.05)
Border: rgba(255,255,255,0.10)
Primary text: #F8FAFC
Secondary text: #94A3B8
Muted text: #64748B
Engineering accent: #38BDF8
AI / research accent: #8B5CF6
Venture accent: #10B981
Warning / experiment accent: #F59E0B
Archived / inactive accent: #EF4444
```

## Typography

Use:

* Strong modern sans-serif for headings
* Clean readable sans-serif for body text
* Monospace for system labels, badges, metadata, and technical tags

Suggested:

* Headings: Inter, Geist, Satoshi, Space Grotesk
* Body: Inter or Geist Sans
* Technical labels: JetBrains Mono or Geist Mono

## Design Elements

Use:

* Glassmorphism cards
* Subtle grid background
* Thin borders
* Glow accents
* Animated status indicators
* Modular dashboard cards
* Architecture diagrams
* Timeline tracks
* Expandable project cards
* Smooth section transitions
* Framer Motion animations
